## 3. 타이포그래피

### 폰트
```
Pretendard → -apple-system → BlinkMacSystemFont → system-ui → sans-serif
```
- CDN: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.css`
- `tailwind.config.js`에서 `fontFamily.sans`로 오버라이드

### 크기 체계
| 요소 | Class | 사이즈 |
|------|-------|--------|
| Hero H1 | `text-4xl sm:text-5xl lg:text-6xl font-extrabold` | 36~60px |
| Page H2 | `text-2xl sm:text-3xl font-bold` | 24~30px |
| Section H3 | `text-xl font-bold` | 20px |
| Card Value | `text-3xl font-extrabold` | 30px |
| Body | `text-base` | 16px |
| Label | `text-sm font-semibold` | 14px |
| Small/Caption | `text-xs` | 12px |

### 한국어 최적화
- `word-break: keep-all` — 단어 단위 줄바꿈
- `tracking-tight` — 제목 글자 간격 조밀
- `leading-relaxed` — 본문 라인 높이 여유

## 10. 핵심 원칙 요약

1. **Utility-First**: 모든 스타일링은 Tailwind 유틸리티 클래스로 작성 (커스텀 CSS 최소화)
2. **White & Blue Edge**: 순백색 배경 + 브랜드 블루 포인트로 신뢰감 전달
3. **Rounded & Soft**: 모든 UI에 `rounded-xl ~ rounded-3xl` + 은은한 `shadow-sm` 적용
4. **Mobile-First**: 기본은 모바일 레이아웃, `md:` 이상에서 2단 그리드 확장
5. **Micro-Interaction**: 호버 시 `-translate-y`, `scale`, `shadow` 변화로 생동감 부여
